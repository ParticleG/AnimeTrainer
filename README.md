# EXplorer

An exhentai api bot based on CQ.

## Requirements

```shell
pip install -r requirements.txt
```

## Start API-backend

Runtime(Production) environment:

```
uvicorn <class-name>:app
```

Or hot-edit:

e.g.

```
uvicorn web-interface-demo:app --reload
```
